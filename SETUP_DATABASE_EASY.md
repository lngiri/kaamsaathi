# 🚀 SETUP DATABASE - COPY PASTE ONLY (2 Minutes)

## Step 1: Open Supabase
- Go to: https://app.supabase.com
- Click your project

## Step 2: Open SQL Editor
- Look LEFT SIDE of screen
- Click **"SQL Editor"** (looks like this icon: <>)
- Click **"New Query"** button (top right)

## Step 3: COPY THE SQL
Open this file in VS Code:
- `supabase/migrations/002_create_all_tables.sql`

Select all (Ctrl+A) → Copy (Ctrl+C)

## Step 4: PASTE in Supabase
- In the SQL Editor query box, Paste (Ctrl+V)
- See all the SQL code? Good!

## Step 5: RUN IT
- Click the **"Run"** button (or press Ctrl+Enter)
- Wait 10 seconds...
- See "✓ Success"? YOU'RE DONE! ✅

## Step 6: Restart your app
```bash
npm run dev
```
